# このソフトウェアについて #

Draw201608141429は私個人が学習目的で作成したソフトウェアです。

描画Frameworkを作成中。
描画アーキテクチャと描画内容のclassを実行時に切り替えることができる。
[前回](https://github.com/ytyaru/Draw201608121536)より少しだけ改善した。
`GraphicsSelector::Select`関数内のif文と文字リテラルをなくした。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
